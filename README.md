## 🚀 Quick Start

### Build and Run
```bash
# Build compliant image
docker build --platform linux/amd64 -t cerberus-ultimate .

# Run with perfect compliance
docker run --rm -v $(pwd)/input:/app/input -v $(pwd)/output:/app/output --network none cerberus-ultimate
```
